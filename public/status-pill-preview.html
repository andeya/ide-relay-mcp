顶栏状态胶囊（示意）

开发时打开：http://127.0.0.1:5173/status-pill-preview.html（需先 npm run dev 或 npm run tauri dev）。结构与 App.vue 顶栏一致：hue-hub 仅光晕脉冲、无 呼吸圈；hue-loading / hue-idle / hue-active 左侧有 statusPillWaitRing（opacity 呼吸）且 idle/active 另加整颗胶囊呼吸。

说明：status === null（拉取 read_tab_status 前）时 statusLabel 仍用 statusAwaiting，故 loading 行文案与 active 相同（ME），仅背景色为 loading。正式 UI 里长文案截断宽度由 .mainTopBarMid（max-width: min(72vw, 560px)）约束；本 预览为单文件，将同等 cap 写在 .mainTopBarStatusPill 上。
